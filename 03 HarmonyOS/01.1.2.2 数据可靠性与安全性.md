# 数据可靠性与安全性
- 存储损坏、存储空间不足、文件系统权限、系统掉电等都可能**导致数据库发生故障**

解决方案
- 备份、恢复功能
- 数据库加密功能，是对整个数据库文件的加密
- 数据库分类分级
    - 基于**数据安全标签和设备安全等级进行访问控制**，保证数据安全。
    - 数据安全标签
        - 将数据分为S1(低)、S2(中)、S3(高)、S4(严重)四个安全等级
    - 设备安全等级
        - 分为SL1、SL2、SL3、SL4、SL5五个等级
        - 手表通常为低安全的SL1设备，手机、平板通常为高安全的SL4设备
- E类加密数据库
    - 目的是在设备**锁屏**状态下通过加密保护敏感数据，防止未经授权的访问

跨设备同步访问控制机制
- 数据库的数据安全标签不高于对端设备的设备安全等级时，数据才能同步

| 设备安全级别 | 可同步的数据安全标签 |
|--------------|----------------------|
| SL1          | S1                   |
| SL2          | S1~S2                |
| SL3          | S1~S3                |
| SL4          | S1~S4                |
| SL5          | S1~S4                |
