## build-profile.json5 中配置项 externalNativeOptions
```
"apiType": "stageMode",
"buildOption": {
  "arkOptions": {
   },
  "externalNativeOptions": {
    "path": "./src/main/cpp/CMakeLists.txt",    #CMake构建脚本地址
    "arguments": "",        # CMake编译参数
    "cppFlags": "",         # C++编译器参数
    "abiFilters": [         # 本机的ABI编译环境
       "arm64-v8a",
       "x86_64"
    ],
  }
}
```
