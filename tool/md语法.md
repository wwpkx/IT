# 公式语法
- **行内公式**:用 `$`包裹公式
- **独立公式**:用`$$`包裹公式
- 上标表示：`^`
- 下标表示：`_`
- 加边框：`\boxed`用于给公式
- **根号**：通用表达方式为`$\sqrt[a]{b}$`→ $\sqrt[a]{b}$
- **分式**
	- `$\frac {a}{b}$` →   $\frac {a}{b}$
	- ``$a+1 \over b+1$`→ a + 1 b + 1 \rightarrow {a+1 \over b+1}`

# 特殊转义字符
`# $ & ~ _ ^ \ { } %`
